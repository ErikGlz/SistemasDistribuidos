Programa cliente-servidor que permite que se envien números enteros entre si.

Programas que se deben ejecutar:
  -Servidor: Servidor
  -Cliente: Cliente_Socket.java
  
El servidor se compila con "make" y se ejecuta con "./Servidor", usar puerto 15557
El cliente se compila con "javac Cliente_Socket.java" y se ejecuta con "java Cliente_Socket"

Para finalizar la conexión el cliente debe enviar un 0 al servidor
